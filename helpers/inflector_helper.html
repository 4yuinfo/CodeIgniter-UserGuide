Inflector 補助函數

The Inflector 補助函數 file contains functions that permits you to change words to plural，singular，camel case，etc.

載入 補助函數

This helper is loaded using the following code:

$this->load->helper('inflector');

The following functions are available:

singular()

Changes a plural word to singular. 參考範例:

$word = "dogs";
echo singular($word); // Returns "dog"

plural()

Changes a singular word to plural. 參考範例:

$word = "dog";
echo plural($word); // Returns "dogs"

To force a word to end with "es" use a second "true" argument.

$word = "pass";
echo plural($word，TRUE); // Returns "passes"

camelize()

Changes a string of words separated by spaces or underscores to camel case. 參考範例:

$word = "my_dog_spot";
echo camelize($word); // Returns "myDogSpot"

underscore()

Takes multiple words separated by spaces and underscores them. 參考範例:

$word = "my dog spot";
echo underscore($word); // Returns "my_dog_spot"

humanize()

Takes multiple words separated by underscores and adds spaces between them. Each word is capitalized. 參考範例:

$word = "my_dog_spot";
echo humanize($word); // Returns "My Dog Spot"
